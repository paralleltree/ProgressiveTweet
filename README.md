# ProgressiveTweet

ProgressiveTweetは、[G2U](https://github.com/0V)氏作成の[AssistiveTweet](https://github.com/0V/AssistiveTweet)に触発されて開発されたTwitterクライアントです。

## 内容
 * やっぱり[アレ](http://support.apple.com/ja-jp/HT202658)っぽいUIです。
 * 現在はテキストのみのツイートしかできません。
